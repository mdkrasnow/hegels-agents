# Documentation Index

This directory contains comprehensive documentation for the Hegels Agents project.

## Directory Structure

### `/api/` - API Documentation
- **Core Interfaces**: Agent protocols, debate interfaces, corpus retrieval
- **Configuration**: Settings, environment variables, and configuration schemas
- **Utilities**: Helper functions and utility classes

### `/guides/` - User and Developer Guides
- **Installation Guide**: Detailed setup instructions for all environments
- **User Guide**: How to use the system for various tasks
- **Developer Guide**: Deep-dive into architecture and extension points
- **Configuration Guide**: Complete configuration reference
- **Troubleshooting**: Common issues and solutions

### `/research/` - Research Documentation
- **Methodology**: Research methods and evaluation frameworks
- **Benchmarks**: Benchmark descriptions, baselines, and results
- **Experiments**: Experiment designs and findings
- **Philosophical Foundation**: Deep-dive into Hegelian dialectics

## Documentation Standards

Our documentation follows these principles:

1. **Progressive Disclosure**: Basic info first, details on demand
2. **Example-Driven**: Code examples for all concepts
3. **Research-Grade**: Reproducible methodologies and results
4. **Maintainable**: Documentation that evolves with code
